Для работы требуется Qt 5.4, работу на других версиях не проверял.
Достаточно установить его и указать путь к .pro файлу (он лежит в папке ESOINlib)
Далее всё должно запуститься и работать по идее.
Работает на WINDOWS и LINUX, но почему-то на виндовс рандом генератор криво работает. Если 
исправить то перестанет нормально работать на линуксе.
Код прокомментирован на английском, прокомментированы самые основные вещи.
В написании использовался C++ + QML
Класс Person взят из какого-то учебного примера, существует он лишь для того, чтобы подать данные
из C++ в QML, потом подправлю название, уберу лишние поля и изменю формат подачи данных.
Сейчас занимаюсь загрузкой изображений в него и доделкой того что сделал.


ВНИМАНИЕ:
1) могут быть большие проблемы при изменении кода: иногда QT забывает обновить makefile, поэтому при любой
ошибке линковки стоит удалить все мейкфайлы в папке, автоматом сгенерированной qt (папки генерируются
с названием компилятора, например виндовский компилятор сгенерировал папку 
build-ESOINlib-Desktop_Qt_5_4_0_MSVC2012_OpenGL_32bit-Debug у меня и там лежит мейкфайл. 
в линуксе папка другая)
2) Если будут проблемы с запуском на винде, возможно стоит удалить эту папку
build-ESOINlib-Desktop_Qt_5_4_0_MSVC2012_OpenGL_32bit-Debug и запустить проект снова.

Вроде всё)
